pixelboook安装Ubuntu16.04

发表于 2018-08-23

上周作死，入手一台pixelbook，使用Chrome OS吧，感觉太鸡肋，所以准备装Linux

本来想装CentOS，但是查了一圈资料没找到crouton安装CentOS的方法，我也算是计算机小白，加之pixelbook价格比较–贵，不敢用来折腾，所以决定安装Ubuntu

插嘴一句，不知是我RP问题还是资料有问题，我参照网上现有资料安装，总能出现各种各样的bug，结合我一周以来每天2点以后才睡觉的痛苦经历，我决定把安装过程记录下来，万一哪天需要重装系统的话，还可以用来参考。

进入开发者模式

  • 同时按住 esc、刷新键还有关机键
    • 当cb屏幕出现恢复模式的提示语，按ctrl+d，等待cb重启
    • 以后每次cb开机都需要按ctrl+d进去系统

下载crouton

  • crouton下载地址

在cb上按ctrl+alt+t 进入crosh

  • 输入shell
  • cd ~/Downloads
  • sudo sh -e ./crouton
  • sh ./crouton -r list //查看crouton支持的版本
    • Ubuntu各版本代号
    • 我安装的是Ubuntu16.04版本，所以就是xenial

开始安装

  • sudo sh -e ~/Downloads/crouton -r xenial -t core
  • sudo sh -e ~/Downloads/crouton -r xenial -t core,audio,cli-extra,core,touch,,x11,xorg,gtk-extra,keyboard,gnome,gnome-desktop -u
    • 我用的是gnome桌面，如果大家想用别的桌面可以自行更改，将gnome,gnome-desktop换成对应的桌面就可以

进入Ubuntu

  • sudo startgnome

Ubuntu跟chrome os切换

  • 同时按下shift+ctrl+alt+刷新(第一排第三个)

在Ubuntu下可能存在的问题

进去以后，按ctrl+alt+t如果没有出现终端，可以尝试一下方法(当初我遇到这个问题困扰了好久,据说是因为语言环境造成的)，参考

  • 首先注销Ubuntu返回chrome os
  • 然后再Ubuntu下进入shell(ctrl+alt+t，输入shell)
  • 在chrome os进入Ubuntu终端
    • sudo enter-chroot
    • sudo apt-get install language-pack-zh-hans language-pack-zh-hans-base language-pack-gnome-zh-hans language-pack-gnome-zh-hans-base
    • sudo apt-get install `check-language-support -l zh`
    • sudo localectl set-locale LANG=zh_CN.UTF-8
    • logout
  • 好了，接下来就可以再次切换到Ubuntu系统中了
    • sudo startgnome

结尾

最后，祝大家chromebook玩的开心
